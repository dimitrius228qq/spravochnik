Добро пожаловать, это сайт-справочник с процессом регистрации и аутентификации
Краткое руководство:
index.php- форма регистрации и аутентификации
admin.php- страница администратора
person.php-страница других пользователей
В директории uploads хранятся изображения абонентов
library/css/style1.css- стилизация для формы регистрации и аутентификации
library/css/stylead.css- стилизация для страницы администратора
library/css/styleper.css-стилизация для страцы других пользователей
library/js/jquery.form.js-библиотека JavaScript
library/js/jquery.min.js-библиотека JavaScript
library/js/main.js-скрипт для страницы администратора
library/js/mainperson.js-скрипт для страницы пользователей
library/php/ajaxdob.php-добавление абонента об абоненте
library/php/ajaximagedel.php-удаление абонента
library/php/ajaxizm.php- вывод информации об абоненте в input
library/php/ajaxizmeneniya.php-изменение данных абонента
library/php/ajaxvid.php-показ данных об абоненте
library/php/auth.php-процесс аутентификации пользователей
library/php/db.php-подключение к базе данных
library/php/index.php-процесс регистрации пользователей
library/php/logout.php- процесс выхода из пространства пользователя
library/php/spisok.php-вывод отсортированного списка всех абонентов на странице администратора
library/php/spisokper.php- вывод списка абонентов на странице пользователя

Для того, чтобы зайти на страницу администратора , необходимо:
1)Зарегистрировать пользователя
Логин : admin
Все остальные поля заполняются поусмотрению
